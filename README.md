# War Brasil
É um projeto no qual buscamos desenvolver uma releitura do jogo de tabuleiro "WAR". Colocaremos, no War Brasil, os estados brasileiros em guerra. Cada jogador iniciará o jogo com territórios e objetivos aleatórios. Será necessário ter estratégia, ter conhecimento das regiões brasileiras e um pouco de sorte para conquistar o objetivo e vencer a partida!
